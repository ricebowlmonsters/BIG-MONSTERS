# 🚀 PANDUAN CEPAT OPERASIONAL KASIR "BIG MONSTERS" 🚀

*(Instruksi: Cetak halaman ini di kertas A4 dan tempel di area Kasir untuk panduan staf)*

---

## ☀️ 1. PERSIAPAN AWAL HARI (SEBELUM BUKA TOKO)
Pastikan 3 langkah ini selalu dilakukan setiap pagi sebelum menerima pesanan pertama:

1. **Nyalakan Hardware:** Nyalakan Komputer Kasir dan SEMUA Printer Thermal (Printer Kasir, Printer Dapur, dan Printer Minuman).
2. **Cek Kertas Struk:** Pastikan kertas thermal di masing-masing printer masih cukup dan tidak terbalik.
3. **JALANKAN SISTEM PENGHUBUNG (PRINT BRIDGE):** 
   * Cari file `start-print-bridge.bat` di komputer.
   * Klik 2x (Double-click) file tersebut.
   * **SANGAT PENTING:** Sebuah jendela layar hitam akan muncul. **JANGAN DITUTUP (X)** jendela hitam tersebut selama toko beroperasi! Anda hanya boleh mengecilkan (minimize) jendela tersebut.

---

## 🍔 2. CARA KERJA CETAK OTOMATIS (SAAT ADA PESANAN)
Aplikasi BIG MONSTERS bekerja secara pintar dan otomatis tanpa perlu klik tambahan:

* **Input Pesanan:** Masukkan pesanan makanan & minuman dari pelanggan di layar kasir seperti biasa.
* **Proses Otomatis:** Saat Anda mengklik "Simpan Pesanan" atau "Bayar", sistem akan otomatis membagi tugas:
  * 🧾 **Printer DAPUR** langsung mencetak rincian pesanan Makanan agar koki bisa langsung masak.
  * 🧾 **Printer BAR** langsung mencetak rincian pesanan Minuman.
  * 🧾 **Printer KASIR** mencetak struk tagihan total untuk diberikan kepada pelanggan.
* *Anda tidak akan melihat jendela pop-up/dialog print di layar komputer. Semuanya berjalan mulus di latar belakang!*

---

## ⚠️ 3. BANTUAN CEPAT (JIKA TERJADI KENDALA)

**Masalah: Struk Tidak Keluar Sama Sekali?**
* **Cek Jendela Hitam:** Periksa apakah jendela hitam "Print Bridge" tertutup/hilang? Jika ya, klik 2x lagi file `start-print-bridge.bat`.
* **Cek Jaringan:** Pastikan kabel LAN / koneksi WiFi printer terpasang dengan baik dan lampu indikator menyala.

**Masalah: Struk Hanya Keluar di Kasir, Dapur Tidak Keluar?**
* Pastikan Printer Dapur dalam keadaan "ON" dan kertas tidak habis/macet.
* Pastikan alamat IP Printer Dapur belum berubah.

**Masalah: Aplikasi Kasir Error atau Loading Terus?**
* Refresh halaman browser aplikasi kasir Anda dengan menekan tombol **F5** di keyboard.
* Pastikan status koneksi internet/LAN di komputer Anda lancar.
